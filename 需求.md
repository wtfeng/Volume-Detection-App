写一个基于安卓，可以用于安卓8.1以上的系统，然后可以检测声音的大小，特别是对人声说话的大小，如果检测到超过一个阈值，就播放一段预先配置好的音乐，然后这段音乐就以配置好的音量（初始配置为最大的声音）来播放。播放音乐的时候就停止检测，等音乐播放完之后再去检测。

本机已经安装好安卓的环境，编译时要先找一下。

这个程序需要尽量不被系统关闭，一直在后台运行。

adb tcpip 43125
adb connect 192.168.1.4:43125

adb install app/build/outputs/apk/debug/app-debug.apk

adb logcat -c

# 查看应用状态
adb shell dumpsys package com.example.volumedetection

# 查看实时日志
adb logcat | grep volumedetection

# 如果应用无响应，强制重启
adb shell am force-stop com.example.volumedetection
adb shell am start -n com.example.volumedetection/.MainActivity
